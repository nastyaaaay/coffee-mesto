## О проекте

Учебный проект-портфолио: одностраничный адаптивный сайт кофейни, вдохновлённый реальным заведением «Место» в Вологде. Не официальный сайт кофейни — сделан в качестве практики вёрстки.

Сайт собран с нуля на чистых **HTML, CSS и JavaScript**, без фреймворков и сборщиков — весь код можно открыть и прочитать напрямую.

## Что сделано

- 🎨 **Дизайн и вёрстка** — фирменная цветовая палитра (крем/эспрессо/карамель), пара шрифтов Fraunces + Manrope, сетки на CSS Grid и Flexbox
- 📱 **Адаптивность** — отдельные брейкпоинты под планшет и мобильные, бургер-меню на маленьких экранах
- 🖱️ **Интерактивность на JS** — переключение вкладок меню (кофе / лимонады / выпечка), смена фона хедера при скролле, мобильное меню
- ✨ **Анимации** — плавное появление блоков при скролле через `IntersectionObserver`, hover-эффекты на карточках и кнопках
- 🗺️ **Секции сайта** — герой, о кофейне, особенности, меню с ценами, отзывы/рейтинг, контакты со ссылкой на карту
- 🚀 **Публикация** — задеплоено на GitHub Pages

## Структура

```
coffee-mesto/
├── index.html      # разметка всех секций сайта
├── styles.css      # стили, переменные темы, анимации, адаптив
├── script.js       # табы меню, бургер-меню, scroll-эффекты
└── assets/         # изображения интерьера и позиций меню
```

## Стек

`HTML5` · `CSS3 (Grid, Flexbox, custom properties)` · `Vanilla JavaScript` · `Google Fonts`
